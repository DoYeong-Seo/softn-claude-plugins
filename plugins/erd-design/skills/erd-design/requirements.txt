#   - pip3 (확인: `pip3 --version`)
#
#   미설치 시 (Ubuntu/WSL):
#     sudo apt update && sudo apt install -y python3 python3-pip
#
# [2] Python 패키지 설치
#     pip3 install -r "${CLAUDE_PLUGIN_ROOT}/skills/erd-design/requirements.txt"

#   - 토큰 발급: https://back.softn.kr → 마이페이지 → PAT 발급
#   - ~/.profile 에 다음 한 줄 추가 후 Claude Code 재시작:
#       export BLOGN_PAT_TOKEN=softn_pat_xxxxxxxx...
#   - 주의: `export` 키워드 없이 추가하면 자식 프로세스로 전파되지 않음.
#   - ~/.bashrc 에 두면 비대화형 셸(Claude Code Bash 툴 포함)에서 로드되지
#     않으므로 .profile 권장.
